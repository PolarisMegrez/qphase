# qphase

`qphase` 是一个用于量子光学相空间模拟的模块化仿真框架。它提供了一个统一的命令行接口 (CLI) 和基于插件的架构，用于管理模拟生命周期、配置和数据持久化。

本项目是一个个人维护的、科研导向的小型工具，旨在促进量子光学领域数值实验的可重复性。

## 核心特性

*   **插件架构**: 框架建立在由协议（Protocols）定义的严格插件系统之上。求解器、计算后端和分析工具与核心逻辑解耦，允许灵活扩展。
*   **配置管理**: 模拟通过结构化的 YAML/JSON 配置文件定义。系统支持参数扫描，并使用 Pydantic 模型进行验证。
*   **任务调度**: 内置调度器管理模拟任务，处理执行顺序、依赖解析和结果聚合。
*   **后端抽象**: 计算后端通过接口进行抽象，使得在不修改核心模拟逻辑的情况下可以使用不同的数值库（如 NumPy, Numba）。
