# SDE 建模指南

QPhase SDE 采用三层架构来建模物理系统。这种层次结构允许在物理定义（量子/相空间）和数值实现（SDE）之间分离关注点。

## 三层架构

### 第一层：主方程 (ME)
**类：** `qphase_sde.model.MasterEquation`

使用哈密顿量 ($\hat{H}$) 和 Lindblad 塌缩算符 ($\hat{L}_k$) 在希尔伯特空间中描述系统。这是最基本的物理描述。

*   **用例：** 从第一性原理定义物理。
*   **组件：** 哈密顿量、Lindblad 算符。

### 第二层：相空间模型 (FPE)
**类：** `qphase_sde.model.PhaseSpaceModel`

在相空间（例如 Wigner、P-表示、Q-函数）中描述系统动力学。它由 Fokker-Planck 方程 (FPE) 的 Kramers-Moyal 展开系数定义。

$$ \frac{\partial P}{\partial t} = \sum_{n=1}^\infty \frac{(-1)^n}{n!} \frac{\partial^n}{\partial \alpha^n} [D_n(\alpha) P] $$

*   **用例：** 解析推导、研究相空间分布。
*   **组件：** 漂移向量 ($D_1$)、扩散张量 ($D_2$)，以及可能的更高阶项 ($D_3, \dots$)。

### 第三层：随机模型 (SDE)
**类：** `qphase_sde.model.DiffusiveSDEModel`、`qphase_sde.model.JumpSDEModel`

描述用于数值仿真的随机轨迹。这是仿真引擎消耗的层级。

*   **DiffusiveSDEModel（朗之万）：** 适用于仅有一阶和二阶项的系统（高斯噪声）。
    $$ d\mathbf{y} = \mathbf{a}(\mathbf{y}, t) dt + \mathbf{b}(\mathbf{y}, t) d\mathbf{W} $$
*   **JumpSDEModel：** 适用于将更高阶项映射到跳跃过程的系统。

## 工作流程

1.  **定义物理：** 首先定义一个包含漂移 ($D_1$) 和扩散 ($D_2$) 系数的 `PhaseSpaceModel`（第二层）。
2.  **转换：** 使用 `qphase_sde.model.fpe_to_sde()` 将 FPE 模型自动转换为 `DiffusiveSDEModel`（第三层）。
3.  **仿真：** 将第三层模型传递给 `Engine`。

或者，高级用户可以直接定义 `DiffusiveSDEModel`，以便手动控制噪声分解。

## 示例：Van der Pol 振荡器

### 第二层定义
```python
from qphase_sde.model import PhaseSpaceModel

def drift_fn(y, t, p):
    # ... 计算 D1 ...
    return d1

def diffusion_fn(y, t, p):
    # ... 计算 D2 ...
    return d2

fpe_model = PhaseSpaceModel(
    name="vdp_fpe",
    n_modes=1,
    terms={1: drift_fn, 2: diffusion_fn},
    params={...}
)

    转换

    from qphase_sde.model import fpe_to_sde

sde_model = fpe_to_sde(fpe_model)
    ```
